# Расширение для сбора ссылок с сайтов(Название этого расширения в файле(Chrome))

## Как использовать?
1. Установите расширение в браузер (Chrome, Yandex, Edge).  
2. Перейдите на сайт, с которого хотите собрать ссылки.  
3. Нажмите на иконку расширения в панели браузера.  
4. Нажмите кнопку «Скачать ссылки!» внизу расширения.  
5. Ссылки будут сохранены в файл links.txt (максимум 15 ссылок).

## Примечания:
- Можно загружать несколько файлов с ссылками — программа автоматически объединит и обработает их все (файлы должны быть в папке «Загрузки»).  
- Чтобы добавить ссылку вручную, просто откройте любой .txt файл с ссылками и впишите её туда.

## Требования

- Go 1.24.0 и выше  
- Браузеры: Chrome, Yandex, Edge  
- Папка «Загрузки» для хранения файлов с ссылками
