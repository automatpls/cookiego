# Расширение для сбора ссылок с сайтов(Название этого расширения в файле(Chrome))

## Как использовать?
1. Добавьте расширение в браузер (Chrome, Yandex, Edge).
2. Откройте сайт, с которого хотите собрать ссылки.
3. Нажмите на иконку расширения.
4. Нажмите на кнопку снизу "Cкачать ссылки!"
5. Готово. Ссылки сохраняются в текстовый файл links.txt(максимум 15 ссылок). 

## Примечания:
- Можно спокойно загружать несколько файлов с ссылками(Программа объединит и обработает все, главное сохранить их в загрузках(Downloads))
- Если хотите добавить отдельно ссылку, просто зайдите в любой файл txt файл с ссылками и введите ее туда
