# Расширение для сбора ссылок с сайтов(Название этого расширения в файле(Chrome.Zip))

## Как использовать?
1. Добавьте расширение в браузер (Chrome, Yandex, Edge).
2. Откройте сайт, с которого хотите собрать ссылки.
3. Нажмите на иконку расширения.
4. Нажмите на кнопку снизу "Cкачать ссылки!"
5. Готово. Ссылки сохраняются в текстовый файл links.txt(максимум 15 ссылок). 

## Примечания:
- Если сайт не открывается, включите VPN.
- Расширение собирает только ссылки, которые есть в коде страницы (динамические ссылки могут не попадать в список).
